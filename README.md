## Setup & Build

```shell
# Install
make install

# One time build
make

# Test with html-proofer
make test

# Build and run locally
make run
```

or use our [Dockerfile](docker/) container

```shell
docker-compose up       # Now you can access the website at localhost:4000/TUM-Projekte/
```
